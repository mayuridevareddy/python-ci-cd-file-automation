## 📌 Overview
This project demonstrates a **CI/CD pipeline using GitHub Actions** to automate the execution of a Python script.  
It highlights basic DevOps practices like automation, workflow configuration, and continuous integration.

---

## ⚙️ Features
✨ Automated pipeline triggered on every code push  
✨ Python script execution using GitHub Actions  
✨ File handling operations (read/write)  
✨ YAML-based workflow configuration  

---

## 🛠️ Tech Stack
- 🐍 Python  
- ⚙️ GitHub Actions  
- 📄 YAML

## 🔄 CI/CD Workflow
1. Code is pushed to GitHub  
2. GitHub Actions triggers the workflow  
3. Python environment is set up  
4. Script runs automatically  
5. Output is displayed in logs
